# HLS Streaming

## Description
This project demonstrates a basic setup for HTTP Live Streaming (HLS).
It is useful for understanding how HLS works using a simple HTML interface
and Docker-based configuration.

## Files Overview
- `index.html` – Basic frontend for streaming
- `commands.txt` – Commands related to HLS setup
- `Dockerfile` – Docker configuration for running the project
